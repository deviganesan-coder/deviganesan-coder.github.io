HS Code Tagger

Associated with Accenture in India

Developed an automated HS Code Tagger to streamline the process of classifying product descriptions into appropriate Harmonized System (HS) codes. This tool was designed to assist in accurate and efficient product classification, improving manual verification processes.

Technical Summary: The HS Code Tagger utilized the eBay API to gather product descriptions and employed Word2Vec embeddings to estimate textual similarities between product descriptions and HS code descriptions. This approach enabled accurate tagging of products based on their descriptions.

Challenge & Solution: Matching product descriptions with HS codes required handling diverse and often ambiguous text data. Employed Word2Vec embeddings to capture semantic similarities, improving the accuracy of the tagging process and reducing errors.

Impact and Outcomes: The tool reduced manual classification time by consolidating all relevant data into a streamlined process, enhancing operational efficiency. Improved accuracy in product classification by leveraging advanced text processing techniques.

Automatic Financial Reconciliation

Associated with Accenture in India

This project aimed to streamline the traditionally tedious and error-prone manual process of matching the ledger records with bank statements. Developed a genetic algorithm-based module that significantly improved the efficiency of the reconciliation process.

Technical Summary: Genetic Algorithms (GAs) are optimization techniques inspired by natural evolution, commonly used for solving complex problems through processes like selection, crossover, and mutation. In this project, a novel GA approach was specifically designed for the automatic reconciliation matching problem. Unlike traditional GA methods that rely on binary or real-valued vector representations, the proposed algorithm uses an innovative arithmetic operation-based crossover directly on floating-point numbers. This novel approach eliminates the need for vector encodings and enhances time efficiency. Additionally, the proposed approach features a tailored selection strategy and mutation operation, resulting in improved accuracy, reduced memory consumption, and faster computation compared to conventional methods.

Outcomes The new module enhanced operational efficiency by reducing the time required for reconciliation, leading to faster financial processing and fewer manual errors. Achieved approximately 60% reduction in running time compared to the previous Java-based module using combinatorial enumeration.
